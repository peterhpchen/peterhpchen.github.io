新增說明文字
這時候有兩個方式可以解決：
1. 改名(例：origin改成first)
2. 將origin的url改掉

第一個方式

把
git remote add origin [url]
改成
git remote add first [url]

第二個方式

利用set-url改掉origin的url
git remote set-url origin [url]
